# Cosmoedu 최종 비즈니스 모델

**작성일**: 2026-01-15
**전략 방향**: 저작권 리스크 제거 + B2B SaaS 집중
**핵심**: 무료 문제은행 + 유료 학원/학교 솔루션

---

## 🎯 최종 전략 확정

### 핵심 원칙

✅ **저작권 있는 시중 문제는 유료 버전에서 절대 사용 안 함**
✅ **B2B SaaS가 주 수익원** (월 ₩55,000 ~ ₩110,000)
✅ **학생 무료, 교사/센터장 유료** 구독 모델
✅ **강사 교육 + 멤버십 혜택**으로 차별화

## 📊 2가지 버전 구조

### Version 1: 무료 버전 (Community Edition)

**타겟**: 모든 학생 (B2C)
**가격**: 완전 무료

#### 제공 기능

```
[문제 은행 - 무료 콘텐츠만]
✅ 공공 대회 기출 문제 (KOI, USACO, ICPC 등)
✅ 교육용 오픈 라이선스 문제
✅ Cosmoedu 자체 제작 문제
✅ AI 생성 문제 (저작권 없음)
   → 총 1,000개 문제 무료 제공

[학습 기능]
✅ 기본 진도 추적 (학습 일수)
✅ 2D CT 그래프 시각화
✅ 문제 풀이 및 채점 (Judge0)
✅ 기본 통계 (정답률, 시도 횟수)

[제한 사항]
❌ AI 맞춤 문제 추천 없음
❌ 3D CT Galaxy 없음
❌ 페르소나 분석 없음
❌ 학습 로드맵 없음
❌ 교사 대시보드 접근 불가
```

**목적**:
- 학생 유입 (리드 생성)
- 데이터 수집 (AI 학습)
- 브랜드 인지도 확보

---

### Version 2: 유료 SaaS (Professional/Enterprise)

**타겟**: 학원/학교 교사 및 센터장 (B2B)
**가격**: ₩55,000 ~ ₩110,000/월

#### 제공 기능

```
[콘텐츠]
✅ 무료 문제 1,000개 (Version 1과 동일)
✅ 유료 자체 제작 콘텐츠 (저작권 100% 소유)
   - 프리미엄 커리큘럼 (Scratch/Entry/Python)
   - 프로젝트 기반 학습 자료
   - 교사용 수업 가이드
   - PPT/워크시트 제공

[진도 관리 시스템]
✅ Teacher Dashboard
   - 학생별 학습 진도 실시간 모니터링
   - CT 5대 역량 분석
   - 학습 시간/빈도 추적
   - 취약점 자동 탐지

✅ 학부모 리포트 자동 발송
   - 주간/월간 학습 리포트
   - CT 성장 그래프
   - 추천 학습 경로

[진로 설계 시스템]
✅ 페르소나 분석 (4가지 유형)
   - 설계자형 (Architect)
   - 해결사형 (Hacker)
   - 표현가형 (Artist)
   - 탐험가형 (Explorer)

✅ 맞춤형 로드맵 (Tech Tree)
   - 학생별 최적 학습 경로
   - AI 기반 다음 단계 추천
   - 진로 연계 콘텐츠 (게임개발, AI, 웹 등)

✅ 3D CT Galaxy 시각화
   - 학생의 CT 신경망 3D 시각화
   - 실시간 활성화 상태 확인

[AI 기능]
✅ 맞춤 문제 추천
✅ 코드 리뷰 AI
✅ 오답 노트 자동 생성
✅ 학습 패턴 분석

[멤버십 혜택]
✅ 월 1회 강사 교육 (온라인/오프라인)
✅ 교사 커뮤니티 접근
✅ 신규 콘텐츠 우선 제공
✅ 1:1 기술 지원
```

### VS 백준/정올 (무료 플랫폼)

| 항목           | 백준/정올                | Cosmoedu                   |
|----------------|--------------------------|----------------------------|
| 타겟           | 개인 학생                | 학원/학교 (B2B)            |
| 수익 모델      | 광고/기부                | SaaS 구독료                |
| 핵심 가치      | 문제 수 (20,000개)       | **진도 관리 + 진로 설계**  |
| 교사 기능      | 없음                     | **Teacher Dashboard**      |

| 학부모 리포트  | 없음                     | **자동 발송**              |

### VS 프로그래머스 (B2C + B2B)

| 항목           | 프로그래머스             | Cosmoedu                   |
|----------------|--------------------------|----------------------------|
| 타겟           | 성인 취업 준비생         | 초중등 학생                |
| B2B 모델       | 기업 채용 연계           | **학원/학교 SaaS**         |
| 콘텐츠         | 코딩테스트 중심          | **CT 기반 교육과정**       |
| 교육 지원      | 없음                     | **강사 교육 + 멤버십**     |
| 가격           | 무료/강의 판매           | **월 구독 SaaS**           |

### 핵심 차별화

> **"백준은 학생을 위한 플랫폼, Cosmoedu는 선생님을 위한 플랫폼"**

```
백준/정올: 학생이 혼자 공부
Cosmoedu: 선생님이 학생을 가르침 + 관리

백준/정올: 진도율 %만 표시
Cosmoedu: CT 5대 역량 + 페르소나 + 로드맵

백준/정올: 교사 기능 없음
Cosmoedu: Teacher Dashboard + 학부모 리포트

백준/정올: 수익 모델 불명확
Cosmoedu: B2B SaaS 명확한 수익
```

## 🎯 최종 요약

### 비즈니스 모델

```
[무료 버전]
- 학생 누구나 무료
- 공공 문제 + 자체 제작 문제 1,000개
- 기본 학습 기능
→ 목적: 리드 생성 + 브랜드 인지도

[유료 SaaS]
- 교사/센터장 구독 (₩55,000 ~ ₩110,000/월)
- 진도 관리 + 진로 설계 + AI 기능
- 강사 교육 + 멤버십 혜택
→ 목적: 주 수익원 + 고객 충성도
```

### 저작권 전략

```
✅ 공공 문제 (KOI, USACO 등)
✅ 자체 제작 (100% 소유)
✅ AI 생성 (저작권 없음)
✅ 커뮤니티 기여 (라이선스 동의)

❌ 시중 문제 (백준, 프로그래머스 등)
❌ 교재 문제
❌ 기업 코딩테스트
```

### 차별화

```
백준/정올: "20,000개 문제를 학생에게"
Cosmoedu: "진도 관리 + 진로 설계를 선생님에게"
```
