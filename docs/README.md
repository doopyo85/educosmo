## 🔗 정책 문서 요약

### 플랫폼 저장소 통합 정책 (핵심)

```
┌─────────────────────────────────────────────┐
│              통일 영역                        │
│  • S3 경로: users/{userID}/{platform}/...    │
│  • API 응답 형식                              │
│  • 프론트엔드 UI                              │
│  • quotaChecker 용량 관리                     │
└─────────────────────────────────────────────┘
                    ↓
┌─────────────────────────────────────────────┐
│              분리 영역                        │
│  • 플랫폼별 DB 테이블                         │
│  • 플랫폼별 API 엔드포인트                    │
│  • 플랫폼별 파일 처리 로직                    │
└─────────────────────────────────────────────┘
```

### S3 경로 규칙

```
users/{userID}/{platform}/{saveType}/{fileName}

예시:
users/student001/entry/projects/my_game_20251218.ent
users/student001/scratch/projects/animation.sb3
users/student001/python/projects/calculator.py
```
