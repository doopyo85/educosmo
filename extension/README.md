# 코딩앤플레이 학습도우미 확장프로그램

코딩앤플레이 학습 플랫폼에서 Entry, Scratch, App Inventor 프로젝트를 제출할 수 있도록 도와주는 Chrome/Edge 확장프로그램입니다.

## 설치 방법

### Chrome에서 설치

1. 이 폴더를 다운로드하여 원하는 위치에 압축 해제합니다.
2. Chrome 브라우저를 열고 주소창에 `chrome://extensions` 를 입력합니다.
3. 우측 상단의 "개발자 모드"를 켭니다.
4. "압축해제된 확장 프로그램을 로드합니다" 버튼을 클릭합니다.
5. 압축 해제한 이 폴더를 선택합니다.
6. 설치 완료!

### Edge에서 설치

1. 이 폴더를 다운로드하여 원하는 위치에 압축 해제합니다.
2. Edge 브라우저를 열고 주소창에 `edge://extensions` 를 입력합니다.
3. 좌측 하단의 "개발자 모드"를 켭니다.
4. "압축을 푼 항목 로드" 버튼을 클릭합니다.
5. 압축 해제한 이 폴더를 선택합니다.
6. 설치 완료!

## 사용 방법

1. 코딩앤플레이(https://app.codingnplay.co.kr)에 로그인합니다.
2. 과제를 선택하고 "프로젝트 열기" 버튼을 클릭합니다.
3. 공식 에디터(Entry, Scratch, App Inventor)가 새 탭으로 열립니다.
4. 프로젝트를 완성합니다.
5. 우측 상단에 표시되는 "제출하기" 플로팅 버튼을 클릭합니다.
6. 안내에 따라 프로젝트 파일을 제출합니다.

## 지원 플랫폼

- **Scratch** (https://scratch.mit.edu)
  - 공유된 프로젝트: 자동 추출
  - 비공유 프로젝트: .sb3 파일 업로드

- **Entry** (https://playentry.org)
  - .ent 파일 업로드

- **App Inventor** (https://ai2.appinventor.mit.edu)
  - .aia 파일 업로드

## 문제 해결

### 확장프로그램이 작동하지 않아요
- Chrome/Edge의 확장프로그램 관리 페이지에서 확장프로그램이 활성화되어 있는지 확인하세요.
- 페이지를 새로고침하세요.

### 제출 버튼이 보이지 않아요
- 코딩앤플레이에서 "프로젝트 열기" 버튼을 클릭하여 에디터를 열었는지 확인하세요.
- 직접 에디터 사이트에 접속한 경우에는 제출 버튼이 표시되지 않습니다.

### 파일 업로드가 실패해요
- 파일 크기가 50MB 이하인지 확인하세요.
- 올바른 파일 형식(.sb3, .ent, .aia)인지 확인하세요.
- 네트워크 연결을 확인하세요.
